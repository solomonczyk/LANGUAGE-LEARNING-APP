# Language Learning App — Master Product Canon

**Status:** CANONICAL  
**Version:** 1.0.0  
**Effective date:** 2026-06-10  
**Owner:** Product Owner  
**Change control:** Changes require a documented change request or ADR where specified.

## Purpose

This package closes the project as a managed product from the current state through the first paid pilot and public MVP decision. It consolidates product, market, commercial, delivery, quality, financial, operational, and governance boundaries.

## Canonical outcome

The team must be able to move from the current repository state to first paying users without reopening fundamental questions about:

- primary customer;
- problem and value proposition;
- MVP cut line;
- commercial offer;
- pricing hypotheses;
- acquisition channels;
- roadmap and deadlines;
- stage budgets;
- success, hold, pivot, and stop criteria;
- quality and safety gates;
- ownership and change control.
