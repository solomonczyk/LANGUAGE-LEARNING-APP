3. Personal narrative is the primary content source
4. CEFR 2020 is the reference framework including mediation and plurilingualism

## Theoretical foundations

### Input Hypothesis (Krashen)
Learners acquire language through comprehensible input — messages they understand. Our lessons provide abundant input through audio narratives, reading texts, and teacher/peer models, all calibrated to the learner's level + 1.

### Output Hypothesis (Swain)
Production pushes learners to process language more deeply than comprehension alone. Every lesson requires meaningful output — speaking and writing that communicates real meaning.

### Interaction Hypothesis (Long)
Negotiation of meaning — clarifying, confirming, repairing — drives acquisition. Our dialogue tasks create opportunities for authentic interaction where meaning must be negotiated.

### Sociocultural Theory (Vygotsky)
Learning occurs in the Zone of Proximal Development with appropriate scaffolding. Our dynamic scaffolding system operationalizes this.

## Core approach: Task-Based Language Teaching (TBLT)

Lessons follow a TBLT cycle:
1. **Pre-task** — Introduction to topic and communicative goal
2. **Task** — Learner performs the communicative task (with scaffolding)
3. **Focus on form** — Attention drawn to linguistic features arising from the task
4. **Practice** — Controlled practice of identified items
5. **Transfer** — New task applying learned items in a different context

## Personal narrative as primary source

Personal stories provide:
- **Affective engagement** — Emotional connection aids memory
- **Rich context** — Stories provide natural communicative purpose
- **Repetition with variation** — Same story retold with increasing sophistication
- **Self-concept investment** — Learner cares about expressing their identity accurately

## Depth of processing

Activities promote deep processing:
- Elaboration (connecting new to known)
- Generation (creating personal examples)
- Evaluation (judging correctness and appropriateness)
- Retrieval (recalling from memory)

## Error treatment philosophy

Errors are:
1. **Diagnosed** — Type, source (L1 transfer, overgeneralization), pattern
2. **Prioritized** — Global (affects meaning) before local (does not affect meaning)
3. **Treated** — Appropriate feedback type selected based on error, learner, context
4. **Tracked** — Added to error evidence model for spaced repetition
